# Explainable Loan Approval Prediction using SHAP and LIME

## Project Overview
This project predicts loan approval using Random Forest and explains the predictions using SHAP and LIME.

## Features
- Random Forest Classification
- SHAP Summary Plot
- SHAP Waterfall Plot
- LIME Explanation

## Technologies
- Python
- Scikit-learn
- SHAP
- LIME
- Pandas
- Matplotlib
